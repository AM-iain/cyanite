Cyanite stores your metrics.

Cyanite provides a metric data-store and API targetted at compatibility with the graphite ecosystem

Get started Learn more

Scalable

By relying on Apache Cassandra, cyanite makes scaling your metric infrastructure a breeze. No more shuffling whisper data around, just add nodes to your cluster!

Fast

The underlying model in cyanite is optimized for very fast writes, and a constant write cost as you scale the cluster up. While this is mostly thanks to Apache Cassandra, cyanite architecture which leverages a fast async event driven network library ensures it doesn't become a bottleneck

Compatible

Fell in love with graphite-web's UI or maybe you're more of a grafana person ?
Either way, cyanite gets you up and running quickly, thanks to its compatibility with graphite and graphite-api.

Simple

Installing and scaling cyanite is 100% worry-free. Relying on a simple YAML configuration file, cyanite will discover and keep track of its cassandra storage cluster as well as maintain a search index of metric names.
